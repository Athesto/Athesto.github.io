Descripción General

Este codigo convierte el resumen de las respuestas en un formato JSON, CSV o objeto de javascript

Descripción detallada

Este codigo viene con 2 funciones principales: getData que toma la información del codigo y data2CSV que recibe un array de objetos y los convierte en un string para csv.
Adicionalmente hay una variable preseteada llamada output que tiene las siguientes propiedades

  • output.json: resumen en formato JSON
  • output.data: resumen como objeto de JS
  • output.csv: string de csv que puedes copiar en un excel o google spreadsheet
  • output.isNotCorrect: muestra solo los valores equivocados
